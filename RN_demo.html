Renault-Nissan Demo

Built ADAS surround view application software based on an existing framework for a demo project for automobile giant Renault-Nissan.

The application was speicifically designed for Renault-Nissan's best-selling SUV, X-Trail. The project was focused rather challenging in terms of the components and timelines Some major task carried out were the following :

• Gain familiarity with architectural design of Texas Instruments Multi core SOC TDA2X (ARM Cortex A15 and Cortex M4 + C66x DSP + EVE cores).

• Board bring of the SoC (Pad config. and CCS & JTAG flashing)

• Developed I2C drivers on top of TI TDA2X framework

• Design control code for Serializer & Deserializer (SerDes)

• HMI implementation

• Fix stability issues
