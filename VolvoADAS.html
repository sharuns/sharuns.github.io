Volvo ADAS systems

On going project for developing a C++ based ADAS Application on the Texas Instruments (TI) ADAS SoC TDA1X platform. Combined with new gen Valeo camera system powered by latest SerDes offering form Maxim integrated

The project involves adapting a brand new camera system with latest components as in OVL image sensor and Maxim integrated SerDes pair.And rewriting the SoC framework to accomadate the new components. Major tasks perfromed were as follows:

• Develop I2C drivers from scratch on top of TI platform framework

• Bring up the forward & backchannel for a new camera system

• Develop drivers for new 967XX SerDes devices from MAXIM

• Design and implement Host Command protocol adaption APIs for Imager and SPI flash communication for OmniVision Imager ISP

• Implement new RID and DID requests

• Conduct multiple debug sessions with part suppliers, architects and reviewers
